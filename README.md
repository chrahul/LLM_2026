# What This GitHUbRepo Actually Teaches

Unlike many AI REPO that stay theoretical, this one is **very practical and code-driven**.

It teaches how LLMs work **and how to use them in real systems**.

### 1. Foundations of LLMs

* Tokens and tokenization
* Embeddings
* Transformers architecture
* Attention mechanism
* Pretraining vs Fine-tuning

### 2. Working with Embeddings

* Semantic search
* Document similarity
* Clustering
* Vector databases

### 3. Prompt Engineering

* Few-shot prompting
* Instruction tuning
* Prompt design patterns

### 4. Fine-Tuning Models

* Fine-tuning LLMs with datasets
* LoRA and parameter-efficient tuning
* Domain adaptation

### 5. Building LLM Applications

* Retrieval Augmented Generation (RAG)
* Chatbots
* Knowledge assistants
* Document Q&A systems

### 6. Evaluating LLMs

* Model evaluation
* hallucination detection
* benchmarking
