# Natural Language Processing (NLP)

## Concept

**Natural Language Processing (NLP)** is a field of Artificial Intelligence that focuses on enabling computers to **understand, process, and generate human language**.

In simple terms, NLP allows machines to interact with humans using natural language like **English, Hindi, Spanish, or any other language**, instead of only numbers or programming commands.

NLP sits at the intersection of three major disciplines:

* **Computer Science** — building algorithms and systems
* **Linguistics** — understanding how human language works
* **Artificial Intelligence** — enabling machines to learn patterns from data

When these three fields come together, we get technologies that allow computers to **read text, understand meaning, and generate responses**.

Today, NLP powers many of the tools we use daily, such as:

* translation systems
* chatbots
* search engines
* voice assistants

Examples include systems like **Google Translate**, **Siri**, and **Alexa**.

---

### Common NLP tasks

NLP systems are used to solve a wide range of language-related problems, including:

* **Machine Translation**
  Converting text from one language to another.

* **Sentiment Analysis**
  Determining whether text expresses positive, negative, or neutral sentiment.

* **Speech Recognition**
  Converting spoken language into text.

* **Text Summarization**
  Producing shorter versions of long documents.

# Historical Development of NLP

Understanding NLP also requires looking at how the field evolved.

### 1950s — Early experiments

The earliest NLP research focused on **machine translation**. Researchers tried to translate languages using rule-based systems and handcrafted grammar rules.

These early systems were limited because human language is extremely complex.

---

### 1980s–1990s — Statistical methods

Researchers shifted toward **statistical approaches**.

Instead of manually writing rules, computers started learning patterns from large text datasets using probability and statistical modeling.

This was the beginning of **data-driven language processing**.

---

### 2010s — Deep learning revolution

The next major shift happened with **deep learning**.

Neural networks started outperforming traditional statistical models.

Finally, in 2017, the introduction of **transformer architectures** changed the entire field.

The famous research paper **Attention Is All You Need** introduced transformers, which later powered models like **BERT** and **GPT**.

These models can understand language context much better than previous approaches.

# Core Techniques Used in NLP

Most NLP systems process language through several stages.

### 1. Tokenization

The text is split into smaller units called **tokens**.

Example:

```
"I love artificial intelligence"
```

Tokenized into:

```
["I", "love", "artificial", "intelligence"]
```

---

### 2. Part-of-Speech Tagging

Each word is assigned a grammatical role.

Example:

```
I (pronoun)
love (verb)
AI (noun)
```

---

### 3. Syntactic Parsing

The system analyzes how words relate to each other in a sentence.

This helps the system understand sentence structure.

---

### 4. Semantic Interpretation

At this stage, the system tries to determine the **meaning of the sentence**.

Example:

```
"Apple released a new phone"
```

Here, *Apple* refers to the company, not the fruit.

---

### 5. Discourse Analysis

This step focuses on understanding meaning across multiple sentences or paragraphs.

For example:

```
John bought a car. He loves it.
```

The system must understand that **“it” refers to the car**.

---

# Modern NLP Approaches

Modern NLP systems often combine all these steps using neural networks.

Two key ideas power modern NLP:

### Embeddings

Embeddings are **vector representations of words** that capture their meaning.

Words with similar meanings appear close together in vector space.

Example:

```
King ≈ Queen
Doctor ≈ Nurse
```

---

### Attention Mechanisms

Attention allows a model to focus on important words in a sentence.

Example:

```
The animal didn't cross the road because it was tired.
```

The model must understand that **“it” refers to the animal**, not the road.

Attention mechanisms allow models to capture such relationships.
